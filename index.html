Controls

  • Left - A / ←
  • Right - D / →
  • Soft drop - S / ↓
  • Hard drop - Space
  • Rotate - R
  • Pause - Esc

About

Simple Tetris clone using my html div-grid.

Some features like "Wall kicks", "Shadow Tetromino", "hold piece", complicated score counting (T-spins etc.) are not implemented because of my boredom with this project.

I tried to stick to the official Tetris guidelines how I could. Nevertheless, I don't want to expand this project furthermore as I reached my goals and want to move on.
